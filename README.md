# huntjs

## Instalação / Uso

### Node - API
<pre><code>
docker pull mongo
docker run --name mongodb -p 27017:27017
cd node-api
npm install
npm run dev
</code></pre>

Ps: Para uso do insomnia, existe o arquivo insomnia.json para import.
